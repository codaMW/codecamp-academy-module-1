# Challenge Exercise: Parking Fee System

## Description

A parking garage wants to determine how much a customer must pay depending on the **type of vehicle** and the **number of hours parked**.

You must create a program that evaluates the parking fee based on these conditions.

Your program must use **nested conditionals**.

---

## Inputs

Your program must ask the user to enter:

- Vehicle type (`car` or `motorcycle`)
- Number of hours parked

---

## Rules

### Step 1 — Validate the inputs

- If the number of hours is **less than or equal to 0** → print **"Invalid parking time"**.
- If the vehicle type is not **"car"** or **"motorcycle"** → print **"Invalid vehicle type"**.

---

### Step 2 — Parking fees

#### If the vehicle is a **car**:

- If the parking time is **2 hours or less** → **$5**
- If the parking time is **more than 2 hours** → **$10**

#### If the vehicle is a **motorcycle**:

- If the parking time is **2 hours or less** → **$3**
- If the parking time is **more than 2 hours** → **$6**

---

## Example Input

Vehicle type: car  
Hours parked: 3

---

## Output

Parking fee: $10

---

### 2. JavaScript
